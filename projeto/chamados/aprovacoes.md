# Fluxo de aprovações

## Regra exclusiva para departamento de operações
**Manobrista abre chamado de compras ou manutenção**
- Encarregado precisa aprovar;
- Em seguida, após o encarregado aprovar, o supervisor precisa aprovar;
- Em seguida após o supervisor aprovar, o gerente precisa aprovar;
- Quando o gerente aprovar, o ticket entra como aberto para o departamento de destino (Compras ou Manutenção por exemplo);

## Regras para departamento de Compras e Manutenção
- Todos podem abrir chamados de compras ou manutenção;
- O chamado não precisa de aprovação para seguir, ja entra com o status "em aberto" nos departamentos de destino para serem tratados;
